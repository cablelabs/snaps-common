# snaps-common

snaps-common is a library of shared utilities for use by SNAPS projects

## Getting started
```Bash
git clone https://github.com/cablelabs/snaps-common.git
pip install snaps-common/
```

If you're new to git and GitHub, be sure to check out the [Pro
Git](https://git-scm.com/book/en/v2) book. [GitHub
Help](https://help.github.com/) is also outstanding.
